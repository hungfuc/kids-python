Let user enter four numbers for two trianglers' base and height. <br/>
Compare the areas of these two triangles. <br/>
Show which one is bigger at the end. <br/>
Show **"Both are the same"** when two areas are equal. <br/>
<br/>
Input and Output:
```
Please enter the first triangle's base
2
Please enter the first triangle's height
3
Please enter the second triangle's base
5
Please enter the second triangle's height
4
```
Output:
```
The second triangle is bigger.
```
