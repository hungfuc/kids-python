Create a menu that shows two options for users to print some outputs. <br/>
The menu looks like the following:
```
Please enter 1, or 2 for the following options:
1. Print a pattern multiple times
2. Exit the program
```

After the user enters option 1, the program prompts the following request for further inputs.
```
Please enter the pattern
```
The user can enter a patter, for example, @_@ <br/>
Then the program prompts another request for user input about how many times.
```
How many times?
```
After the user enter 5, the output will be as follows:
```
@_@ @_@ @_@ @_@ @_@
```
Then, the menu will show again to ask user to choose the menu option
```
Please enter 1, or 2 for the following options:
1. Print a pattern multiple times
2. Exit the program
```
If the user enter 2, the terminate the program and show.
```
Program Ends!
```
The following is an example of program execution.
